# 153-gf-calculator
CS 153 Project on Galois Field Calculator by Mary Abigail del Castillo

Language used: Python 2.7
1) Make sure to at least have Python 2.7
2) There is an option to run the source code through the command: python delcastillo_gf_calcu.py
OR 
Run the executable file: 
a) For Windows: click on delcastillo_gf_calcu.exe
b) For Linux: 
	1) Open the terminal on the same folder as delcastillo_gf_calcu file
	2) Run by the command: ./delcastillo_gf_calcu
